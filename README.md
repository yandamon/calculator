This is a simple calculator with basic operations.

###
Simply click the link https://damon-calculator.herokuapp.com/ 

###
OR

###
Install the project by running the command below one by one

###
git clone https://github.com/yandamon/calculator

###
npm install

###
npm start

###
Note: in this application, square and square root are included.
The square operation is "^", and the square root is placed by "?" for easier input.
